# Evaluaci-n3_IA
En este repositorio se implementará un agente funcional completo, para un proyecto llamado MediSearch, capaz de orquestar herramientas de consulta, escritura y razonamiento en un flujo de trabajo organizacional automatizado. En el cual se destaca la implementación de los siguientes factores:
Observabilidad básica
Trazabilidad avanzada
Seguridad y ética
Escalabilidad y sostenibilidad
